# Transitio-user-css

我的 [Transitio](https://github.com/PRO-2684/transitio) user css.

## [all-in-one](./all-in-one.css)

大杂烩：包括以下除了 `demo-mode` 和 `hide-self` 外的所有样式。

隐藏一些不需要的项目，详见源码注释。
